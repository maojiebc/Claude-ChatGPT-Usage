# Changelog

## 1.1.2 - 2026-07-14

- 修复点击 Claude 浮窗关闭按钮后，`hidden` 状态被持久化导致刷新页面仍不显示的问题。
- 关闭现在只影响当前页面会话；刷新后自动恢复上次的收起态或展开态。
- 已存在的 `hidden` 本地设置会自动迁移为默认收起态，仍可用 `Alt + Shift + U` 即时恢复。
- 鼠标移入 Claude 收起卡片时立即展开，点击继续作为触控与键盘兼容路径保留。

## 1.1.1 - 2026-07-14

- 修复 Claude 将动态额度提示拆成多个文本节点后，完整句式正则无法命中的问题。
- 动态翻译会聚合最小候选容器中的文本，再输出中文提示，同时保留原有 DOM 元素与事件。
- 兼容零宽字符和不换行空格，确保 Fable 5 百分比及重置时间持续汉化。

## 1.1.0 - 2026-07-14

- 按参考设计重构 Claude 用量组件，新增 `collapsed`、`expanded`、`settings`、`hidden` 四种状态。
- 收起态改为 104px 宽的 `5h / 7d / F5` 三行胶囊，只保留额度类型和剩余百分比。
- 展开态改为 304px 紧凑卡片，显示完整名称、剩余进度、剩余时长与 `MM/DD 周X HH:mm` 重置时间。
- 增加悬停提示、点击外部与 `Esc` 收起、离开后自动收起、关闭与恢复入口。
- 增加自动收起延迟、重置时间显示、垂直位置等设置，并通过 `claude-usage-monitor:settings:v1` 持久化。
- Claude 浮窗使用 Shadow DOM 隔离样式，保留现有额度抓取、解析和计算逻辑；ChatGPT 面板行为保持不变。

## 1.0.6 - 2026-07-14

- ChatGPT 收起态与 Claude 统一为周期、剩余百分比和倒计时三层结构。
- 将冗长的“每周使用限额”缩写为 `7d`，避免截断和重复表达。
- 重置卡调整为仅在展开态显示，移除收起态中的额外区块。

## 1.0.5 - 2026-07-14

- ChatGPT 面板增加重置卡摘要，显示当前可用次数。
- 从可用重置卡中计算并显示最近一个过期时间，包含星期和具体时刻。
- 重置卡功能保持只读，不提供卡片消耗操作；总数以接口 `available_count` 为准。

## 1.0.4 - 2026-07-14

- 对齐 ChatGPT 网页端“使用限制”页面，只显示套餐共享的每周使用限额。
- 忽略 `additional_rate_limits` 中 GPT-5.3-Codex-Spark 等内部模型计量项。
- 移除 ChatGPT 面板中的当前、标准、主/次窗口和模型额度等非网页端维度。

## 1.0.3 - 2026-07-14

- 参照 Claude 网页端按额度类别展示的方式，移除 ChatGPT 接口内部模型名和主/次窗口字段。
- 将通用限额显示为 Codex 当前、标准、每周或长周期额度。
- 将 GPT-5.3-Codex-Spark 的独立限额显示为更易理解的“Spark 独立额度”。

## 1.0.2 - 2026-07-14

- 汉化 Claude 首页随时段变化的 Morning、Afternoon、Evening 动态问候语。
- 汉化包含百分比、模型名和重置日期的动态额度提示，并将英文日期时间格式转换为中文。
- 监听文本内容更新，额度百分比或重置时间变化后仍会自动汉化。

## 1.0.1 - 2026-07-14

- 重构 ChatGPT/Codex 展开面板，将模型名、窗口类型和额度周期分层显示。
- 长模型名改为单行省略，完整信息保留在悬停提示中，避免孤字换行和横向溢出。
- ChatGPT 主额度、次额度使用一致的标题与周期徽标，并规范 `Pro Lite` 等套餐名称。

## 1.0.0 - 2026-07-14

- 基于 `jyking/claude2cn` 的 MIT 授权代码建立独立维护版本。
- 保留 Claude.ai 完整中文汉化与 Claude Design 翻译资源。
- 增加 Claude Fable 5 独立周额度显示。
- 兼容新版 `limits[]`、旧版 Fable 字段和多种重置时间格式。
- 增加 ChatGPT/Codex 主窗口、次窗口及额外模型额度显示。
- 增加解析回归测试与 ChatGPT 页面烟雾测试。
