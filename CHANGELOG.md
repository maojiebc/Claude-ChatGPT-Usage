# Changelog

## 1.4.3 - 2026-07-14

- 将设计稿风格的 7 枚生成图标迭代到 Claude 浮窗：闪电、时钟、日历、大脑、刷新、设置与关闭。
- Claude 展开态的额度类型、标题栏和操作区改用透明 PNG 资产；ChatGPT 继续保留自身品牌与票券图标。
- 图标通过带 SHA-256 校验的固定版本资源脚本加载，避免页面 CSP、白底和远程资源漂移。

## 1.4.2 - 2026-07-14

- 工作量档位（Low / Medium / High / Extra / Max）恢复英文显示：产品语境下的强度词，中文直译生硬。
- 「工作量」菜单名、说明句与「默认」徽标保持中文。

## 1.4.1 - 2026-07-14

- 修复 ChatGPT 浮窗"漂移"：水平方向改为永远吸附左/右边缘（默认右缘 8px，与 Claude 的贴边一致），拖动只保留垂直位置与停靠边。
- 拖动松手自动吸附到卡片中心更近的一侧，带 200ms 吸边动画；左停靠时展开卡从左缘向右生长。
- 旧版本保存的任意悬空坐标（如 right: 420px）升级后自动归位贴边，仅继承垂直位置与停靠边。
- 拖动被系统打断（pointercancel）时回弹到停靠位，不再悬停半空；未发生位移的点击不再改写保存的位置。
- 存储格式简化为 `{top, isRight}`，键名不变，向后兼容。

## 1.4.0 - 2026-07-14

- 补齐 Claude 模型选择器与工作量菜单汉化：模型标语（For your toughest challenges 等）、工作量说明与 Low/Medium/High/Extra/Max/Default 档位，动态的 "Included until 日期" 译为「X月X日前可用」；新词条走主脚本增补词典，不动锁定在 v1.0.0 tag 的主词库。
- 额度配色从"按类型固定色"改为"剩余量健康度三档"（沿用原版 claude2cn 阈值）：已用 <60% 绿、<85% 琥珀、≥85% 红；额度类型的区分交给条目图标，颜色专注表达"还剩多少"。
- 整体视觉减重：阴影减淡、收起卡 104→96px、行高 34→30px、展开态百分比 25→22px、图标与徽章缩小并去投影、页眉页脚收紧，观感向原版的轻盈靠拢。
- Claude 与 ChatGPT 两端同步以上配色与减重（共享层一处生效）。

## 1.3.0 - 2026-07-14

- ChatGPT/Codex 面板整体重绘，与 Claude 浮窗统一为同一套设计语言：Shadow DOM 隔离、104px 收起胶囊卡、304px 展开卡、200ms 过渡动画、明暗主题跟随。
- 展开态新增 ChatGPT 品牌绿渐变徽章与套餐徽标（Pro Lite 等），每周使用限额条目配日历图标、品牌绿进度条与大号剩余百分比。
- 重置卡改为独立区块：紫色票券图标 + "N 次可用"徽标 + 最近到期时间，无可用卡时显示 0。
- 保留 ChatGPT 面板既有交互：悬停展开、移出收起、整体拖动（位置仍存 `claude2cn-chatgpt-usage-position`）、触屏 tap 切换。
- 剩余额度 ≤20% 时百分比与进度条转告警红，≤10% 时收起态该行泛红，口径与 Claude 一致。
- 抽出 `widgetSharedStyles()` 共享样式层与 `updateQuotaNodes()` 共享增量渲染，两端视觉后续同步演进不再漂移。
- 明暗主题统一走 `data-theme` 属性，移除旧版内联米色面板样式；倒计时改为增量文本更新，不再整卡重建。
- 预览 harness 支持一键切换 Claude / ChatGPT 站点，mock 对应接口与页面底色。

## 1.2.0 - 2026-07-14

- 按参考稿重绘 Claude 浮窗展开态：每个额度条目新增彩色类型图标（5 小时=时钟、7 天=日历、Fable 5=火花），名称改回主文字色，进度条加粗且暗色下轨道清晰可见。
- 标题栏闪电改为橙色渐变徽章，视觉锚点与参考稿一致。
- 收起态胶囊底色回归中性浅灰，仅百分比着类型色；剩余额度 ≤10% 时该行底色泛红作深度告警。
- 展开/收起/设置面板改为 200ms 缩放淡入淡出过渡，替代原先的瞬间显隐；`prefers-reduced-motion` 下自动禁用。
- 展开卡去掉 306px 最小高度，加载中、获取失败等空态不再撑出大片空白；内容超高时仅列表区内滚，标题与底栏钉住。
- 设置面板增加毛玻璃背景，不再透出下层进度条文字。
- 视口窄于 900px 时始终以收起态启动，避免记忆的展开态遮挡正文。
- 新增 `tests/preview/harness.html` 本地预览环境：mock 额度接口即可在浏览器中真实渲染浮窗，支持明暗主题与正常/低额度/接口失败三种场景切换。

## 1.1.2 - 2026-07-14

- 修复点击 Claude 浮窗关闭按钮后，`hidden` 状态被持久化导致刷新页面仍不显示的问题。
- 关闭现在只影响当前页面会话；刷新后自动恢复上次的收起态或展开态。
- 已存在的 `hidden` 本地设置会自动迁移为默认收起态，仍可用 `Alt + Shift + U` 即时恢复。
- 鼠标移入 Claude 收起卡片时立即展开，点击继续作为触控与键盘兼容路径保留。

## 1.1.1 - 2026-07-14

- 修复 Claude 将动态额度提示拆成多个文本节点后，完整句式正则无法命中的问题。
- 动态翻译会聚合最小候选容器中的文本，再输出中文提示，同时保留原有 DOM 元素与事件。
- 兼容零宽字符和不换行空格，确保 Fable 5 百分比及重置时间持续汉化。

## 1.1.0 - 2026-07-14

- 按参考设计重构 Claude 用量组件，新增 `collapsed`、`expanded`、`settings`、`hidden` 四种状态。
- 收起态改为 104px 宽的 `5h / 7d / F5` 三行胶囊，只保留额度类型和剩余百分比。
- 展开态改为 304px 紧凑卡片，显示完整名称、剩余进度、剩余时长与 `MM/DD 周X HH:mm` 重置时间。
- 增加悬停提示、点击外部与 `Esc` 收起、离开后自动收起、关闭与恢复入口。
- 增加自动收起延迟、重置时间显示、垂直位置等设置，并通过 `claude-usage-monitor:settings:v1` 持久化。
- Claude 浮窗使用 Shadow DOM 隔离样式，保留现有额度抓取、解析和计算逻辑；ChatGPT 面板行为保持不变。
